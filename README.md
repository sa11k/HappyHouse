## 1. 프로젝트 개요

### 🏆 **프로젝트 목표**

### 아파트 매매 거래 정보를 쉽게 검색할 수 있는 서비스입니다.

> 지도를 통해 주변 아파트 정보와 각 아파트의 거래 정보를 확인할 수 있습니다.

1. 관심지역으로 설정한 사용자에게 새로운 매물 등록 안내 문자 전송
2. 관심 매물, 관심 동네 설정
3. 챗봇 (채널톡 활용)
4. 지역 안전 지수 차트 구성(chart.js)
5. 부동산 뉴스 크롤링
6. 법정동으로 아파트 매매 내역 조회
7. 상권 정보 API를 활용한 주변 시설 안내
